## Installation

```bash
git clone --depth=1 https://github.com/princejoogie/next-app-bp.git <project-name>
cd <project-name>
rm -rf .git
git init .
```

## Usage

### Development

```bash
yarn dev # or npm run dev
```

### Test

```bash
yarn test # or npm run test
```

### Build

```bash
yarn build # or npm run build
```
